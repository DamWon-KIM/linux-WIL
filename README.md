# linux-WIL
리눅스 실습 과제 제출용 레포지토리

## Assignments
각자 자신의 이름(혹은 github 닉네임)으로 된 디렉토리를 만들어서, 그 안에 실습 내용을 md 형식으로 작성해서 PR을 보내주세요.

폴더 구조는 아래와 같은데, 본인 폴더만 만드신다면 그 내부 구조는 어떻게 되든 괜찮습니다.
